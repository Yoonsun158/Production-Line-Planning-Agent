# 废旧电池回收预设工艺单元

## U1 - 收集与初步分选
- 功能：从回收渠道收集废旧电池，按类型、品牌、外观状态进行初步分类
- 输入：混合废旧电池
- 输出：按类分组的电池批次
- 适用：所有类型电池（锂离子、铅酸、碱性、镍氢等）
- 设备：分拣传送带、X射线荧光光谱仪(XRF)、条码/二维码扫描器
- 注意事项：操作人员需佩戴绝缘手套，防止短路引发热失控

## U2 - 安全放电
- 功能：对残余电量电池进行安全放电，消除短路和热失控风险
- 输入：带电废旧电池
- 输出：完全放电的电池（开路电压<1V）
- 适用：锂离子电池、镍氢电池、铅酸电池
- 不适用：碱性一次电池（残余电量极低，可直接跳过）
- 设备：电阻放电柜、盐水浸泡池
- 注意事项：需在防爆环境中操作；锂电池放电过程中可能产生热量和气体，需通风

## U3 - 模块拆解
- 功能：拆解电池模组/电池包，分离电芯、BMS电路板、外壳、线束和连接件
- 输入：放电后的电池模组或电池包
- 输出：单体电芯、壳体材料（铝/钢）、电子元件（BMS）、铜线束
- 适用：动力电池（方形/软包/圆柱模组）
- 设备：自动化/半自动化拆解平台、绝缘工具
- 注意事项：方形电池包较规整易拆解；软包电池需防止刺穿铝塑膜；圆柱电池模组点焊连接需专用工具

## U4 - 粗破碎
- 功能：将电芯或电池整体进行初步机械破碎，破碎粒度至10-50mm
- 输入：单体电芯或整只小型电池
- 输出：粗碎物料混合物
- 适用：所有类型电池
- 设备：剪切式破碎机、锤式破碎机
- 注意事项：锂离子电池必须在惰性气氛（N₂/Ar）或水雾条件下操作，防止电解液燃烧；铅酸电池需在负压环境操作防酸雾

## U5 - 细粉碎
- 功能：将粗碎物料进一步研磨至1-5mm，充分解离各组分
- 输入：粗碎物料
- 输出：细碎混合粉料（含铜箔、铝箔、隔膜碎片、黑粉）
- 适用：锂离子电池
- 设备：球磨机、气流磨
- 注意事项：需控制研磨温度，避免黑粉氧化

## U6 - 筛分
- 功能：按粒度分级，分离不同粒径范围的组分
- 输入：细碎混合粉料
- 输出：粗粒级（铜箔、铝箔碎片）、细粒级（正负极黑粉）
- 适用：锂离子电池
- 设备：振动筛、旋转筛
- 分级标准：一般以0.5mm为界分粗细两级

## U7 - 磁选
- 功能：利用磁场分离铁磁性杂质（钢壳碎片、铁钉等）
- 输入：分级或混合物料
- 输出：去铁后物料 + 铁质回收物
- 适用：所有类型电池
- 设备：永磁滚筒分选机、电磁分选机
- 回收价值：铁质材料可直接出售给钢铁回收商

## U8 - 风选
- 功能：利用气流按密度和形态分离轻质组分（隔膜碎片、塑料）和重质组分（金属粉末）
- 输入：混合碎料
- 输出：轻质组分（隔膜、塑料薄膜） + 重质组分（金属粉末、黑粉）
- 适用：锂离子电池、碱性电池
- 设备：Z字形风选机、旋风分离器
- 注意事项：风速需精确控制，过高会带走有价黑粉

## U9 - 比重分选
- 功能：利用物料密度差异进行分选（重力分选/水介质分选）
- 输入：混合物料
- 输出：按密度分级的物料（如铅材料 vs 塑料隔板）
- 适用：铅酸电池（铅板与塑料隔板分离）、锂电池（铜铝箔分离）
- 设备：摇床、跳汰机、重介质旋流器
- 原理：铅密度11.34 g/cm³远高于塑料1.0-1.4 g/cm³，分选效果优异

## U10 - 浮选
- 功能：利用矿物表面亲水/疏水性差异，在水介质中分离石墨与金属氧化物
- 输入：细磨矿浆（黑粉悬浮液）
- 输出：石墨精矿（疏水上浮）+ 金属氧化物精矿（亲水沉底）
- 适用：锂离子电池黑粉处理
- 设备：浮选柱、浮选槽
- 药剂：煤油（捕收剂）、松醇油（起泡剂）
- 注意事项：石墨表面碳涂层需要适当处理才能实现有效浮选分离

## U11 - 黑粉收集
- 功能：收集经分选后的正负极活性物质粉末，即"黑粉"
- 输入：分选后富集的活性物质
- 输出：黑粉（LFP体系含LiFePO₄/石墨；NMC体系含LiNiₓCoᵧMnᵤO₂/石墨）
- 适用：锂离子电池
- 设备：旋风收集器、袋式除尘器
- 黑粉成分：LFP黑粉含Li 3-4%、Fe 30-35%、P 15-18%；NMC黑粉含Ni 15-25%、Co 5-10%、Mn 5-10%、Li 5-7%

## U12 - 湿法冶金
- 功能：通过酸浸、萃取、沉淀等化学方法从黑粉中提取有价金属
- 输入：黑粉或金属富集物
- 输出：硫酸钴、硫酸镍、硫酸锰、碳酸锂等高纯金属盐产品
- 适用：三元(NMC/NCA)、磷酸铁锂(LFP)、碱性电池
- 设备：酸浸反应釜、萃取箱、沉淀过滤系统
- 工艺流程：酸浸（H₂SO₄+H₂O₂）→ 除杂（Fe/Al/Cu）→ 共萃分离（P507萃取剂）→ 沉淀结晶
- 优势：回收率高（>95%），产品纯度高（>99.5%）
- 注意事项：产生酸性废水需配套U15处理

## U13 - 火法冶金
- 功能：通过高温熔炼还原金属，将电池废料中的金属组分在高温下还原为粗金属
- 输入：含金属废料、黑粉或整只电池
- 输出：粗金属锭/合金（如精铅、铜镍钴合金）
- 适用：铅酸电池（回收精铅）、部分锂电池（回收钴镍合金）
- 设备：回转窑、电弧炉、鼓风炉
- 工艺温度：铅酸回收约800-1100°C；钴镍合金约1400-1600°C
- 优势：工艺简单，处理量大，适合混合废料
- 劣势：能耗高，锂随渣损失，无法直接回收锂
- 注意事项：需配套U14尾气处理系统

## U14 - 尾气处理
- 功能：处理破碎、热处理、冶金过程中产生的有害气体和粉尘
- 输入：含HF、SO₂、CO₂、粉尘、有机气体的废气
- 输出：达标排放气体
- 适用：涉及热处理和化学反应的所有工艺段
- 设备：碱液喷淋塔、活性炭吸附塔、布袋除尘器、HEPA过滤器
- 排放标准：需满足GB 16297《大气污染物综合排放标准》
- 关键指标：HF < 5 mg/m³、粉尘 < 20 mg/m³

## U15 - 废水处理
- 功能：处理湿法冶金及清洗过程产生的含重金属和酸碱废水
- 输入：含重金属离子（Ni²⁺、Co²⁺、Mn²⁺、Pb²⁺）、酸碱废水
- 输出：达标排放水 + 含金属固体废渣（送有资质单位处置）
- 适用：湿法冶金(U12)后段、酸浸清洗段
- 设备：中和反应池、沉淀池、压滤机、反渗透膜系统
- 排放标准：需满足GB 8978《污水综合排放标准》
- 工艺：调pH（加NaOH/Ca(OH)₂）→ 重金属沉淀 → 絮凝沉降 → 过滤 → 达标排放

## U16 - 梯次利用评估
- 功能：检测退役动力电池残余容量、内阻和安全性能，评估是否可降级再利用
- 输入：退役动力电池（电芯或模组）
- 输出：分级结果——A级（梯次利用）/ B级（需回收拆解）
- 适用：动力锂电池（LFP/NMC），通常来自新能源汽车退役电池包
- 设备：电池检测系统（充放电循环测试仪、内阻测试仪、绝缘测试仪）
- 判定标准：残余容量 ≥ 80% 额定值 → A级，可用于储能电站、低速电动车、通信基站备电等
- 经济价值：梯次利用电池售价约为新电池的30-60%，经济效益显著
- 注意事项：需逐只/逐模组检测，一致性差的模组不适合梯次利用

## U17 - 材料再生造粒
- 功能：将回收的高纯金属盐重新合成为电池级正极材料前驱体
- 输入：高纯硫酸镍/硫酸钴/硫酸锰溶液或碳酸锂
- 输出：三元前驱体颗粒 Ni₍ₓ₎Co₍ᵧ₎Mn₍ᵤ₎(OH)₂ 或磷酸铁锂前驱体
- 适用：NMC三元材料再生、LFP材料再生、碱性电池锌锰材料再生
- 设备：共沉淀反应釜、喷雾干燥塔、烧结炉
- 工艺：共沉淀 → 陈化 → 洗涤过滤 → 干燥 → 高温烧结 → 造粒
- 产品指标：D50粒径3-12μm，振实密度>1.8g/cm³
- 意义：实现"电池到电池"的闭环回收

## U18 - 最终包装与质检
- 功能：对回收的金属产品和再生材料进行质量检测、计量和包装入库
- 输入：各工艺段的回收产品（金属盐、前驱体、精铅、合金等）
- 输出：符合质量标准的可销售回收产品
- 适用：所有类型电池回收产线的末端工序
- 设备：ICP-OES光谱仪（成分分析）、激光粒度仪、自动包装线
- 质检项目：金属纯度、粒度分布、水分含量、杂质元素含量
- 包装规格：金属盐通常为25kg/袋或1000kg/吨袋
