# 电池技术参考文件

## 磷酸铁锂电池 (LFP)
- 化学体系：LiFePO₄正极 / 石墨负极
- 封装形式：方形铝壳（主流）、圆柱、软包
- 标称电压：3.2V
- 能量密度：140-180 Wh/kg
- 循环寿命：3000-6000次（80% SOH）
- 工作温度：-20°C ~ 60°C
- 主要品牌：宁德时代(CATL)、比亚迪(BYD)、国轩高科、亿纬锂能
- 主要应用：新能源汽车动力电池、储能电站、电动公交
- 正极材料组成：Li 4.4%, Fe 34.4%, P 19.0%, O 42.2%
- 安全特性：热稳定性优异，热失控温度>500°C，不易起火
- 回收价值：中等（锂资源价值为主，铁磷价值较低）
- 回收特点：LFP黑粉中锂含量较低(3-4%)，湿法提锂是关键工艺
- 退役标准：容量衰减至额定80%以下退役，大量适合梯次利用

## 三元锂电池 (NMC)
- 化学体系：LiNiₓCoᵧMnᵤO₂正极 / 石墨负极 (x+y+z=1)
- 常见配比：NMC111、NMC523、NMC622、NMC811
- 封装形式：方形、软包、圆柱(21700/18650)
- 标称电压：3.6-3.7V
- 能量密度：200-300 Wh/kg（NMC811最高）
- 循环寿命：1000-2000次（80% SOH）
- 工作温度：-20°C ~ 55°C
- 主要品牌：宁德时代、LG新能源、三星SDI、松下
- 主要应用：高端电动汽车、高性能储能
- 正极材料组成(NMC622)：Ni 36.2%, Co 12.2%, Mn 11.4%, Li 7.1%, O 33.1%
- 安全特性：高镍型热稳定性较差，热失控温度约200-250°C
- 回收价值：高（含钴、镍等高价值金属，吨回收价值可达数万元）
- 回收特点：镍钴资源回收经济性极佳；高镍(NMC811)钴含量降低但镍价值提升

## 三元锂电池 (NCA)
- 化学体系：LiNi₀.₈Co₀.₁₅Al₀.₀₅O₂正极 / 石墨+硅碳负极
- 封装形式：圆柱(21700/18650)
- 标称电压：3.6V
- 能量密度：250-300 Wh/kg
- 循环寿命：800-1500次
- 主要品牌：松下（特斯拉供应商）
- 主要应用：特斯拉Model S/X/3/Y
- 回收价值：高（镍钴铝，工艺类似NMC）
- 回收特点：圆柱电池数量大（一辆车数千只），拆解工序与方形不同

## 铅酸蓄电池
- 化学体系：PbO₂正极 / Pb负极 / H₂SO₄电解液
- 封装形式：塑料壳体（PP），多格串联
- 标称电压：2.0V/格，12V电池为6格
- 能量密度：30-40 Wh/kg
- 循环寿命：300-500次（深循环型）
- 主要品牌：天能、超威、骆驼、风帆
- 主要应用：汽车启动电池(SLI)、电动自行车、UPS不间断电源、通信基站
- 材料组成(12V 60Ah)：铅及铅合金约60-70%、硫酸约15-20%、塑料壳5-10%、隔板及其他5-10%
- 安全特性：硫酸为强腐蚀性液体；铅为重金属有毒物质
- 回收价值：高（铅回收市场成熟，回收率>95%）
- 回收特点：工艺成熟（火法冶金为主），回收体系完善；需特别注意酸液和铅尘的环保处理

## 碱性锌锰电池
- 化学体系：MnO₂正极 / Zn负极 / KOH电解液
- 封装形式：圆柱（AA/AAA/C/D/9V），钢壳
- 标称电压：1.5V
- 容量：AA型约2500-3000mAh
- 类型：一次电池（不可充电）
- 主要品牌：金霸王(Duracell)、南孚、劲量(Energizer)、松下
- 主要应用：遥控器、手电筒、玩具、钟表、烟雾报警器
- 材料组成：锌粉约15-20%、MnO₂约25-35%、KOH电解液约5-8%、钢壳约25-30%、碳棒及其他10-15%
- 安全特性：风险极低，无热失控风险，KOH有腐蚀性但含量少
- 回收价值：中低（锌锰金属价值一般，但量大）
- 回收特点：无需放电步骤（残余电量极低）；钢壳可磁选回收；锌锰通过湿法冶金分离提纯

## 镍氢电池 (NiMH)
- 化学体系：Ni(OH)₂正极 / 储氢合金(MH)负极 / KOH电解液
- 封装形式：圆柱（AA/AAA）、方形模组
- 标称电压：1.2V
- 能量密度：60-80 Wh/kg
- 循环寿命：500-1000次
- 主要品牌：松下、FDK、比亚迪
- 主要应用：丰田混合动力汽车(HEV)、充电式日用电池、应急照明
- 材料组成：镍约30-40%、稀土合金约20-30%、钴约3-5%、KOH约10%
- 回收价值：中高（含镍和稀土，稀土回收有战略价值）
- 回收特点：KOH电解液需中和处理；储氢合金中稀土(La/Ce/Nd)回收工艺复杂

## 锂聚合物电池 (LiPo)
- 化学体系：LiCoO₂或NMC正极 / 石墨负极 / 凝胶聚合物电解质
- 封装形式：软包（铝塑复合膜）
- 标称电压：3.7V
- 能量密度：200-250 Wh/kg
- 主要应用：智能手机、平板电脑、无人机、可穿戴设备
- 安全特性：软包易膨胀鼓包，刺穿风险导致热失控
- 回收价值：中高（含钴电池价值高）
- 回收特点：软包拆解需防止铝塑膜破裂释放电解液；单体容量小数量大
